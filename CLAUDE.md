# CLAUDE

Read `AGENTS.md` first. This file adds Claude-specific collaboration guidance.

## Planning constraints

- Preserve the single-domain monorepo architecture.
- Assume apps are embedded by default unless the task clearly requires separation.
- Keep v1 English-first and focus on strong landing-page content.
- Treat `apps/tubetrace` as imported source-of-truth code that may be gradually folded into the hub route.

## Editing priorities

- Favor content structure that improves indexing and trust.
- Maintain consistency between app metadata and rendered pages.
- Keep tool pages useful even without client-side hydration.
- Treat guide content as part of the monetization and SEO strategy, not an afterthought.

## Content rules

- Use precise, non-hyped language.
- For uncertain tools, add disclaimers near the top and near the interactive surface.
- Keep copy aligned with privacy-first, browser-based positioning when true.
